## 1. ENVIRONMENT LOCK

```
North African Mediterranean beach, fine white sand with subtle

lapping at shoreline, clear sky with thin high clouds, no visible 
buildings or structures in background, open horizon where pale

```

### Environment DON'TS

- No palm trees (not accurate for Egyptian North Coast)
- No umbrellas, beach chairs, or resort furniture
- No people in background
- No dramatic clouds or storm lighting
- No rocks or cliffs
- No visible villas or buildings (keep it raw/isolated)

## 2. COLOR GRADE LOCK

Apply this to every generation to maintain visual cohesion across the campaign.

```
Warm neutral color grade, slightly desaturated highlights, 
rich but not oversaturated midtones, subtle warm cast in 
shadows, fine organic film grain, natural skin warmth without 
orange shift, whites slightly creamy not pure white, 
turquoise water rendered as natural teal-green not electric blue, 
overall tone: Kodak Portra 400 film stock feel
```

### Color DON'TS

- No HDR / hyper-saturated look
- No cold/blue shadows
- No Instagram-filter-style color grading
- No crushed blacks — keep shadow detail
- No pure white highlights — keep them creamy/warm
- No neon or electric water colors

---

## 3. REALISM RULES — ANTI-AI CHECKLIST

These rules apply to EVERY image. They are what separate a "real" lookbook from an AI render. Mention explicitly in prompts.

### Include (realism signals)

- Visible skin pores and natural skin texture on face and arms
- Subtle sun squint or relaxed eyes — not wide-open "AI eyes"
- Micro-asymmetry in facial expression (one eyebrow slightly higher, slightly uneven smile)
- Natural fabric wrinkles and creases — linen should look lived-in, not pressed
- Wind interaction: hair movement, shirt fabric catching breeze, skirt hem lifting slightly
- Weight distribution: one hip shifted, one foot slightly ahead of the other
- Sand on feet / lower ankles where model has been walking
- Subtle sweat sheen on forehead/neck in daylight shots
- Natural body proportions — no elongated limbs or tiny waists
- Slight shadow under chin and nose from overhead sun

### Exclude (AI tells to avoid)

- Perfectly smooth skin
- Symmetrical poses
- Stiff straight arms at sides
- Dead-center framing
- Uniform flat lighting with no direction
- Clothing that looks ironed/pressed/wrinkle-free
- Perfect hair without any flyaways
- Eyes looking directly into lens in every shot (vary the gaze)
- Background that feels like a flat backdrop

### Per-Batch Workflow

For each batch inside `~/higgsfield-agent/`:

1. Load brain files: NANOBANANA_BRAIN → HIGGSFIELD_BRAIN → Seedance 13-fashion-lookbook
2. Build prompts using the template from Section 6, filling in garment description from Section 7, pose from Section 5, environment from Section 1
3. Show all prompts — wait for approval
4. Generate via CLI: `/higgsfield:generate` with Nano Banana Pro model + character_id, 4K resolution, aspect ratio as specified
5. ONE generation per shot in production phase — save credits for re-dos
6. Save to images/ESM-campaign/ organized by category

### Model Settings

- **Image model:** Nano Banana Pro — primary model for all generations (best product/fashion detail, 4K output)
- **Resolution:** 4K (maximum quality)
- **Aspect ratios:** as specified per shot (4:5, 9:16, 16:9)
- **Character IDs:** Always pass character_id for both models
- **Enhance prompt:** OFF — we are providing precisely engineered prompts, do not let the model rewrite them

### Credit Budget Strategy (2,000 credits total)

With limited credits, we do NOT generate 2-3 variations per shot across all 67 shots. Instead:

**PHASE A — Validation Tests (budget: ~100 credits)**

Run exactly 5 test shots before committing to full production. These test the hardest variables: character accuracy, garment rendering, environment consistency, couple composition.

| Test | What it validates | Shot code |
|------|------------------|-----------|
| TEST-1 | Male character + olive set + beach + daylight (redo of your failed Gen 3, with proper prompt) | LB-M11 |
| TEST-2 | Female character + olive chemise+skirt + beach + daylight (first female generation) | LB-F07 |
| TEST-3 | Male character + white t-shirt + back print detail (tests if branding renders) | LB-M02 |
| TEST-4 | Male character + beige set M-FSB (tests new garment, no product photos) | LB-M15 |
| TEST-5 | Couple shot + beach (tests two characters together) | LB-C03 |

**Evaluate tests against these pass/fail criteria:**
- Character face matches Soul sheet? → if NO, adjust character_id or model
- Garment looks like the product photo? → if NO, refine garment description
- Beach looks like North Coast Egypt? → if NO, refine environment lock
- Pose feels natural, not stiff? → if NO, add more micro-body-language detail
- T-shirt back print renders? → if NO, accept that back-print shots may need inpainting or skip

Only proceed to Phase B after all 5 tests pass quality bar.

**PHASE B — Full Production, 1 shot each (budget: ~864 credits)**

Generate all 48 shots, ONE generation per shot (no variations). This is the most credit-efficient approach. With proper prompts from Phase A learnings, most shots should land on first try.

48 shots × ~18 credits per 4K Nano Banana Pro image ≈ ~864 credits

**PHASE C — Redo & Variations (budget: ~1,500 credits remaining)**

After reviewing all 48, re-generate only the shots that failed (bad pose, wrong garment detail, character drift). Spend remaining credits on:
- Re-dos of failed shots
- Extra variations of the absolute best shots (for Instagram carousel options)
- Additional couple shots if chemistry landed well in Phase B

This approach ensures you never burn credits on a broken prompt template. Phase A catches problems at 5 credits, not 48.

## 11. POST-GENERATION

### Curation Criteria

After each batch, review every shot against these criteria. Flag failures for Phase C re-generation:

1. **Character accuracy** — does it look like the Soul character reference?
2. **Garment accuracy** — does the clothing match the product photos?
3. **Naturalness** — does it feel like a real photo or does it trigger "AI" feeling?
4. **Pose life** — does the body language feel alive and natural?
5. **Light consistency** — does it match the environment lock?
6. **Fabric behavior** — does the linen look like linen (wrinkled, textured, wind-responsive)?

Shots that pass all 6 → selects folder. Shots that fail 1-2 → re-do in Phase C. Shots that fail 3+ → prompt needs rewriting before re-do.
